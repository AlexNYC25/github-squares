# Github Squares

A Stream Deck plugin to display your GitHub contributions graph on your Stream Deck.

Ever wanted to see your GitHub contributions right on your Stream Deck? This plugin fetches your GitHub contribution data and displays it as colored squares, giving you a quick visual representation of your activity.

So if you ever wanted to show off your coding streaks or just keep track of your contributions, this plugin is for you!

All's needed is your Github username and an access token you can generate from your GitHub account settings.

## Features
- Displays your GitHub contributions as colored squares.
- You can customize the number of days to display.
- Easy to set up and use.
- Refreshes data automatically when settings are changed or on demand by pressing the key.

## Technologies Used
- TypeScript
- Stream Deck SDK
- GitHub API (GraphQL)
- Node.js

## Installation

### Manual Installation
1. Download the latest release of the plugin from the [Releases](https://github.com/AlexNYC25/github-squares/releases) page.
2. Install the Stream Deck SDK if you haven't already.
3. Run the build command to compile the plugin 

### Elgato Stream Deck Store
You can also find and install the plugin directly from the Elgato Stream Deck Store.
